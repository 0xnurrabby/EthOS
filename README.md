<div align="center">
<i>A single-page HTML joke site that sends a polite code request to @Ethereum_OS .... the "No" button runs away when you try to click it.</i>
</div>

---

```
+------------------------------------------+
|  Greetings Dear @Ethereum_OS,            |
|  Can I get a code please?                |
|  [ Yes ]   [ No (runs away) ]            |
+------------------------------------------+
```

---

## ✦ Features

<div align="center">

| | Feature | What it does |
|:---:|---|---|
| 😂 | Runaway No button | The "No" button moves away every time you try to click it |
| 🪟 | Modal popup | Clicking "Yes" opens a second confirmation dialog |
| 🎨 | Clean minimal UI | Simple centered card layout with smooth transitions |
| 📦 | Zero dependencies | Pure HTML + CSS + inline JavaScript, no build needed |

</div>

---

## ✦ Download & Run

**Step 1** .... Clone the repo

```bash
# For Eth-OS
git clone https://github.com/0xnurrabby/Eth-OS
cd Eth-OS

# For EthOS (alternate version)
git clone https://github.com/0xnurrabby/EthOS
cd EthOS
```

**Step 2** .... Open in browser

```bash
start index.html
# No server needed
```

**Step 3** .... Done.
